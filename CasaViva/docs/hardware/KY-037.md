# Sensor KY-037 – Detecção de Som

## Descrição
  O KY-037 capta sons fortes e vibrações no ambiente. É ideal para detectar quedas, batidas ou situações anormais.

## Funções no CasaViva
- Gera alerta visual no LCD.
- LED RGB pisca azul durante evento de ruído.
- Pode ser usado para automações baseadas em voz ou eventos abruptos.

## Pinagem
  | Pino | Função | ESP32 |
  |------|--------|--------|
  | A0   | Saída analógica | GPIO 33 |
  | D0   | Saída digital   | (não usado) |
  | VCC  | 3.3V |
  | GND  | Terra |

## Código Referência
```
int som = analogRead(33);
if (som > 2500) triggerAlertaSom();
```
