# Potenciômetro – Ajuste de Threshold

## Descrição
O potenciômetro ajusta dinamicamente o limite ambiental para o MQ-135.

## Pinagem
| Pino | ESP32 |
|------|--------|
| SIG  | GPIO 34 |
| VCC  | VIN |
| GND  | GND |

## Uso
O valor lido controla:
- Limite alto ou baixo de poluição
- Sensibilidade dos alertas
