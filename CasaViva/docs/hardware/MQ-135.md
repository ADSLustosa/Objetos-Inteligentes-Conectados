# Sensor MQ-135 – Qualidade do Ar

## Visão Geral
O MQ-135 detecta compostos gasosos como CO₂, amônia, benzeno e vapores orgânicos. No **Projeto CasaViva**, ele ajuda a monitorar a saúde do ambiente interno e ajustar comportamentos através do LED RGB e possíveis acionamentos automáticos.

## Aplicações
- Identificação de ar poluído em ambientes fechados.
- Automação de ventiladores/umidificadores.
- Alertas quando a qualidade do ar está fora do aceitável.

## Especificações Técnicas
| Propriedade | Valor |
|-------------|--------|
| Tensão      | 5V (no Wokwi ≈ simulado) |
| Sensores    | CO₂, NH₃, vapores tóxicos |
| Tempo de pré-aquecimento | 20–30s |

## Pinagem
| Pino | Função               |
|------|-----------------------|
| VCC  | 3.3V (Wokwi permite) |
| GND  | Terra                 |
| AOUT | Leitura analógica (GPIO35) |

## Integração no CasaViva
- Threshold ajustável pelo potenciômetro.
- LED RGB muda conforme nível:
  - Verde → Adequado  
  - Amarelo → Médio  
  - Vermelho → Ruim  

## Código Referência
```
int value = analogRead(35);
```

### Observações:
    Como o MQ-135 é simulado no Wokwi, usa-se variação manual com controles do simulador.
